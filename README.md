# nextjs-app-starter

A minimal **Next.js 14 + TypeScript + TailwindCSS** starter with App Router, ESLint and Prettier.

## Quickstart

```bash
# 1) Install deps
npm install

# 2) Run dev server
npm run dev
# then open http://localhost:3000

# 3) Build and start production
npm run build
npm start
```

## What’s inside?

- Next.js 14 (App Router) — `app/` directory
- TypeScript configured — `tsconfig.json`
- Tailwind CSS 3 — `tailwind.config.ts`, `app/globals.css`
- ESLint (next/core-web-vitals) — `.eslintrc.json`
- Prettier + plugin for Tailwind class sorting — `.prettierrc`
- Example API route — `app/api/health/route.ts`
- Example component — `components/Card.tsx`
- Example env file — `.env.example`

## Notes

- Put secrets into `.env.local` (not committed).
- Update `metadata` in `app/layout.tsx`.
- Adjust `tailwind.config.ts` content globs if you add new folders.
